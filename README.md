## Features

Slimmed down feature set of original sushii-bot.

* moderation tools
    * ban / unban
    * kick
    * prune
    * mute / unmute
    * cases
        * reason
        * history
    * roles
* user
    * avatar
    * userinfo
    * notifications
    * xp, ranking, activity, leaderboard
    * rank
    * rep
    * fishy
* last fm
* tags
    * add
    * delete
    * random
